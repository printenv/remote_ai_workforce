### Setup Secretes and variables (Actions) bellow on github
- APP_PATH : App's directory path on VPS
- DEPLOY_KEY: SSH private key that matches public key set up on VPS
- HOST: VPS IP address
- PORT: SSH port on VPS
- USERNAME: VPS user name
